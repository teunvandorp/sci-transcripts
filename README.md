# Transcripts for Statistical and Causal Inference

This repository contains lecture transcripts for the course "Statistical and Causal Inference", given as part of the Data Science master program at Radboud University, Nijmegen, by Johannes Textor.

Note that this isn't a textbook; these are edited transcripts of live lectures. Things in here might be inaccurate, vague, or even wrong -- characteristics of natural speech. If you have any feedback, you're more than welcome to raise an Issue or even submit a pull request!
